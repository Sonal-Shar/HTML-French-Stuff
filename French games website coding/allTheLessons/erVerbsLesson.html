Present Tense of Regular Verbs

    Verbs that end in -er
    -take the -er off the ending and add the following endings:

    Je		-e			Nous		-ons
    Tu		-es			Vous		-ez
    il/elle/on	-e			ils/elles	-ent

    Je: Parler -  Parler - Parl + e = Parle	                Nous: Parler -  Parler - Parl + ons = Parlons
    Tu: Parler -  Parler - Parl + es = Parles		        Vous: Parler -  Parler - Parl + ez = Parlez
    il/elle/on: Parler -  Parler - Parl + e = Parle             ils/elles:Parler -  Parler - Parl + ent = Parlent
    
    Verbs that end in -re
    -take the -re off the ending and add the following endings:

    Je		-s			Nous		-ons
    Tu		-s			Vous		-ez
    il/elle/on	-			ils/elles	-ent

    Je: Perdre-  Perdre- Perd + s = Perds		Nous: Perdre- Perdre- Perd + ons =Perdons
    Tu: Perdre-  Perdre- Perd + s = Perds		Vous: Perdre-  Perdre- Perd + ez = Parlez
    il/elle/on: Perdre-  Perdre- = Perd			ils/elles:Perdre- Perdre- Perd + ent = Perdent
    
    Verbs that end in -ir
    -take the -ir off the ending and add the following endings:
    
    Je		-is			Nous		-issons
    Tu		-is			Vous		-issez
    il/elle/on	-it			ils/elles	-issent
    
    Je: Batir -  Batir- Bat + is = Batis			Nous: Batir- Batir- Bat+ issons = Batissons
    Tu: Batir -  Batir- Bat + is = Batis			Vous: Batir- Batir- Bat+ issez = Batissez
    il/elle/on: Batir -  Batir- Bat + it = Batit		ils/elles:Batir - Batir- Bat+ issent = Batissent
